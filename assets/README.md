# Assets Directory

This directory contains all static assets for the Awadh Gully website.

## Structure

- `images/` - Logo, hero images, and other graphics
- `icons/` - Favicons and app icons

## Logo Requirements

### Main Logo
- **File**: `assets/images/logo.png` or `assets/images/logo.svg`
- **Format**: PNG (with transparency) or SVG (preferred for scalability)
- **Recommended Size**: 200x200px minimum, 400x400px optimal
- **Background**: Transparent

### Favicon Assets Needed

For optimal cross-platform support, create the following favicon sizes:

1. **favicon.ico** - 32x32px (multi-size ICO file containing 16x16, 32x32, 48x48)
2. **favicon-16x16.png** - 16x16px
3. **favicon-32x32.png** - 32x32px
4. **apple-touch-icon.png** - 180x180px (for iOS devices)
5. **android-chrome-192x192.png** - 192x192px (for Android)
6. **android-chrome-512x512.png** - 512x512px (for Android)

## Optimization Guidelines

1. **PNG Images**: Use tools like TinyPNG or ImageOptim to compress
2. **SVG**: Prefer SVG for logos (scalable, smaller file size)
3. **WebP**: Consider WebP format for better compression (with PNG fallback)
4. **Favicons**: Use online tools like RealFaviconGenerator or Favicon.io
